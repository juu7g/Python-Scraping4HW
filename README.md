# Python-Scraping4HW

## 概要 Description
ハローワークの求人情報を自動で検索します。  

## 特徴 Features

- 検索条件は設定ファイルで設定します。  
- 結果はCSVファイルに出力します。  
- 結果は50件まで出力します。    
- 出力ファイル名は「月日_時分_秒.txt」です。  

## 依存関係 Requirement

- Python 3.8.5  
- beautifulsoup4 4.9.3    
- selenium 3.141.0    

## 使い方 Usage

こちらのサイトを参照  
[ハローワークの求人情報を自動で検索するアプリ(スクレイピング) - 【非公開】プログラムでおかえしできるかな](https://juu7g.hatenablog.jp/entry/Python/scraping/HWexe)

## インストール方法 Installation

- pip install beautifulsoup4  
- pip install selenium
